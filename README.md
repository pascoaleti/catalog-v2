## Highlights (v2.0)
- 3 catalog modes: **Generic, Real Estate, Vehicles**
- Optional **location fields** (address / region)
- Optional **Blog module** (listing + post page)
- **Home & Blog banners / hero blocks** configurable in Admin
- **Advanced rich-text editor** for items, pages and posts
- **Email sender settings** (From / Reply-To)
- **Category subscriptions with email alerts**
- Remodeled item page with organized contact panel + share links
- SEO helpers: meta tags, Open Graph, canonical, hreflang and JSON-LD (where applicable)
